//// ----- //// Aula sobre 4- Animações //// ----- //// 

Sintaxe:

@keyframes nomeanimação {
	X% {comando;}
	X% {comando;}
}

Uso dela:
.exemplo {
	animation-name:nomeanimação; <<Chama ela
	animation-duration:Xs (seconds); << Duração
	animation-delay: xS	<< tempo pra começar
	animation-iteration-count: 3 <<Quantas vezes repetir; (INFINITE);

}


EXEMPLO DE USO:

Criando:

@keyframes animateste {
	0% { background-color: blue; top:0; left:0;}
	25% { background-color: yellow; top:0; left:300; }
	50% { background-color: black; top:300; left:300;}
	75% { background-color: red; top:300; left:0;}
	100% { background-color: green; top:0; left:0;}
}


.classequalquer {
	width: 200px;
	height: 200px;
	background-color: green;
	position: absolute;
	animation-name:animateste;
	animation-duration:4s;
	animation-delay: 1s;
	animation-iteration-count:infinite;

}

Sintaxe:
.class/#id {

 box-shadow: (distancia da sombra horizontalmente)px (distancia da sombra verticalmente)px (esfumação em PX da sombra)px (Cor da SOMBRA[#888888]); << se for negativo, irá pra esquerda/cima, se não, direita/baixo

//// ----- //// Aula sobre 6- Transições //// ----- //// 


Controlar transições de elementos e mudanças de propiedades. (HOVER) por exemplo

Sintaxe: 
transition-duration: 1s; << Duração que ocorrerá a transição
transition-timing-function: linear; << como se comporta o ritmo da transição durante o efeito  || outros::  linear, ease, ease-in, ease-out e ease-in-out;
transition-delay: 0.1s; << Define a partir de quanto tempo (em segundos) o efeito da transição vai se iniciar (o padrão é 0).

OUU: 

transition: |property| |duration| |timing-function| |delay|;
